Random Data API

Generate random users, grades, and companies on demand. A lightweight, free-to-use REST API for testing, prototyping, and seeding databases.

Explore Endpoints View Source ↗
GET https://random-data-api-vvy8.onrender.com/api/random/users/{count}

02

Authentication

ℹ️
No authentication required.
This API is completely open and free to use. No API key, token, or header is needed. Simply make a GET request to any endpoint and you'll receive data instantly.
03

Error Handling

The API uses standard HTTP status codes to indicate success or failure.

Status Code Meaning Description
200 OK Request succeeded. Body contains the JSON array.
400 Bad Request Invalid {id} parameter. Must be a positive integer.
404 Not Found Endpoint path does not exist.
500 Server Error Unexpected server-side failure.

Random Users

Returns an array of randomly generated user objects. Each user has a name, email address, and a bcrypt-hashed password. Specify how many users you want by replacing {count} with any positive integer.

Path Parameters

Parameter Type Required Description
count Integer Required Number of user objects to generate (e.g. 5 returns 5 users)

Example Request

GET https://random-data-api-vvy8.onrender.com/api/random/users/3

Random Grades

Returns an array of randomly generated student grade records. Each record contains a student ID, a class ID, and an array of scored assessments broken down by type (exam, quiz, homework).

Path Parameters

Parameter Type Required Description
count Integer Required Number of grade records to generate

Example Request

GET https://random-data-api-vvy8.onrender.com/api/random/grades/1

Field Type Description

Code Examples

Integrate the API in your preferred language. All examples fetch 5 random users — swap the endpoint for grades or companies as needed.

JavaScript · Fetch API 
const response = await fetch(
  'https://random-data-api-vvy8.onrender.com/api/random/users/5'
);

if (!response.ok) {
  throw new Error(`HTTP error: ${response.status}`);
}

const users = await response.json();

// Each user: { name, email, password }
users.forEach(user => {
  console.log(`${user.name} — ${user.email}`);
});
JavaScript · Axios 
// npm install axios
import axios from 'axios';

const { data: grades } = await axios.get(
  'https://random-data-api-vvy8.onrender.com/api/random/grades/5'
);

// Each grade: { student_id, class_id, scores: [{type, score}] }
grades.forEach(g => {
  const exam = g.scores.find(s => s.type === 'exam');
  console.log(`Student ${g.student_id} exam score: ${exam?.score}`);
});

Python · grades example 
import requests

BASE_URL = "https://random-data-api-vvy8.onrender.com"

grades = requests.get(f"{BASE_URL}/api/random/grades/5").json()

# Each grade: { student_id, class_id, scores: [{type, score}] }
for g in grades:
    exam = next((s for s in g['scores'] if s['type'] == 'exam'), None)
    print(f"Student {g['student_id']} — exam: {exam['score']:.1f}")
Java · HttpClient (Java 11+) 
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class RandomDataApiExample {

    private static final String BASE_URL =
        "https://random-data-api-vvy8.onrender.com";

    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();

        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(BASE_URL + "/api/random/users/5"))
            .header("Accept", "application/json")
            .GET()
            .build();

        HttpResponse<String> response =
            client.send(request, HttpResponse.BodyHandlers.ofString());

        // [{name, email, password}, ...]
        System.out.println(response.body());
    }
}
Java · Spring RestTemplate 
import org.springframework.web.client.RestTemplate;

public class ApiService {

    private final RestTemplate rest = new RestTemplate();
    private static final String BASE =
        "https://random-data-api-vvy8.onrender.com";

    public Object[] fetchCompanies(int count) {
        // Returns [{name, permalink, categoryCode, offices, ...}]
        return rest.getForObject(
            BASE + "/api/random/companies/" + count,
            Object[].class
        );
    }
}
TypeScript · Typed interfaces 
// --- Users ---
interface User {
  name:     string;
  email:    string;
  password: string;

// --- Grades ---
interface Score {
  type:  "exam" | "quiz" | "homework";
  score: number;
}
interface Grade {
  student_id: number;
  class_id:   number;
  scores:     Score[];
}

// --- Companies ---
interface Office {
  description:  string;
  city:         string;
  state_code:   string | null;
  country_code: string;
  latitude:     number;
  longitude:    number;
}
interface Company {
  name:              string;
  permalink:         string;
  description:       string;
  categoryCode:      string;
  foundedYear:       number;
  numberOfEmployees: number;
  offices:           Office[];
  twitterUsername:   string;
  totalMoneyRaise:   string | null;
}

const BASE = "https://random-data-api-vvy8.onrender.com";

async function fetchUsers(count: number): Promise<User[]> {
  const res = await fetch(`${BASE}/api/random/users/${count}`);
  if (!res.ok) throw new Error(`Error ${res.status}`);
  return res.json() as Promise<User[]>;
}

const users = await fetchUsers(5);

Go · net/http with structs 
package main

import (
  "encoding/json"
  "fmt"
  "net/http"
)

type User struct {
  Name     string `json:"name"`
  Email    string `json:"email"`
  Password string `json:"password"`
}

const BASE = "https://random-data-api-vvy8.onrender.com"

func main() {
  resp, err := http.Get(BASE + "/api/random/users/5")
  if err != nil { panic(err) }
  defer resp.Body.Close()

  var users []User
  json.NewDecoder(resp.Body).Decode(&users)

  for _, u := range users {
    fmt.Printf("%s — %s\n", u.Name, u.Email)
  }
}
